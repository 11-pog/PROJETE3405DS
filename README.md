# Projete 2025 - Equipe 3405

# Read-Cycle

##  Visão Geral

Este projeto tem como objetivo criar uma plataforma digital para **troca e empréstimo de livros** entre usuários, incentivando o compartilhamento de conhecimento, a leitura e o uso consciente de recursos. A aplicação permite que usuários cadastrem livros, solicitem empréstimos ou realizem trocas de forma simples e organizada.

O sistema foi pensado para ser intuitivo, acessível e escalável, servindo tanto para uso pessoal quanto comunitário.

---

##  Funcionalidades

* Cadastro de usuários
* Cadastro de livros disponíveis para troca ou empréstimo
* Visualização do catálogo de livros
* Solicitação de troca ou empréstimo
* Controle de status dos livros (disponível, emprestado, em negociação)
* Histórico de trocas e empréstimos
